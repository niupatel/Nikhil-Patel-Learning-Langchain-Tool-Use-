To install the LangChain package:
LangChain provides integrations to hundreds of LLMs and thousands of other
integrations. These live in independent provider packages.
See the  for a full list of available integrations.
Now that you have LangChain installed, you can get started by following the 
.
npm install langchain @langchain/core
# Requires Node.js 20+
# Installing the OpenAI integration
npm install @langchain/openai
# Installing the Anthropic integration
npm install @langchain/anthropic
